Server Storage Extension

Store and retrieve key/value data from a Server Storage backend, with two editing modes for either live syncing or fast local caching.
Important:
Your API key grants access to all data stored under that key.
If you lose or change it, your existing stored data may become inaccessible.
Create / manage API key (opens [currentURL]/apiKey.html)

API keys

API keys identify your storage “account” on the server. Every request includes your key so the server knows which data belongs to you.
example: paste-your-api-key-here
In PenguinMod, set it once using the extension’s set API key to [APIKEY] block.

Editing modes

Live live
Live mode talks to the server on every read/write:
  • save sends immediately.
  • get fetches the latest server value.
  • delete deletes on the server immediately.
Local local
Local mode caches values in memory and avoids constant networking:
  • save writes to local cache only.
  • get reads from local cache only.
  • Nothing changes on the server until you push.

Using local editing

A simple recommended workflow:
  • Set your API key.
  • Switch to local editing.
  • Run download all keys to local cache to load the current server state into the cache.
  • Do lots of reads/writes locally while your project runs.
  • When ready, run push local changes to server to sync modified keys.

When to use which

Use live if you need the latest shared values immediately (for example, multiple users/projects reading the same data). Use local if you want speed, fewer requests, and smoother gameplay/UI updates while editing or running heavy loops.

Host your own server

If you want full control and reliability, you can host the Server Storage backend yourself and point the extension to it with set server to [SERVER] server. You can host this on any server that has static hosting and a database backend.
The server is open source here: https://github.com/Ikelene/scratchStorageExtension/tree/main
After hosting, generate an API key on your hosted domain, then use that key in PenguinMod.
